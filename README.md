# SchumacherOCR

![logo](/assets/logo.svg)

Библиотека предоставляет модули для работы с изображениями в педагогических 

## Установка

```bash
pip install SchumacherOCR
```

## 

### Работа с изображением

```python
from Schumacher.Ocr import Processor

Processor

```
### Пайплайн обработки

```python
from Schumacher.Ocr import Processor

Processor

```
